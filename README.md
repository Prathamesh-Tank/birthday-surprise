# Birthday Website Specification for Rupinder

## Website Theme

Create a **luxury, elegant, feminine, emotional birthday tribute website** for **Rupinder**, celebrating her birthday on **2nd May**.

The website should feel:

* Warm
* Classy
* Beautiful
* Premium
* Emotionally touching
* Academic yet personal
* Inspired by soft pastel luxury birthday cards
* Designed like a modern storytelling portfolio + birthday tribute

The design inspiration should blend:

* Elegant birthday invitation aesthetics
* Luxury photo scrapbook
* IIT campus memories
* Research-inspired visual storytelling
* Soft feminine minimalism

# Website Type

Single-page immersive scrolling website.

Style:

* Fullscreen sections
* Smooth transitions
* Soft animations
* Luxury typography
* Emotional storytelling flow
* Mobile responsive
* Desktop-first premium experience

Recommended Tech Stack:

* React + TailwindCSS
* Framer Motion for animations
* GSAP for smooth scrolling effects
* Lenis smooth scroll
* React Icons
* Lightbox Gallery

* Glassmorphism cards
